# 机械工业时代 Factory

一个基于原生 HTML/CSS/JavaScript 开发的工业网络建造游戏原型。玩家通过矿源、采矿机、熔炉、铸造厂、组装工厂、仓库、发电机和电杆搭建资源生产网络，完成第一阶段工业目标。

## 运行方式

本项目不依赖 npm 包，直接使用 Node.js 启动本地静态服务器：

```bash
node server.mjs
```

打开浏览器访问：

```text
http://127.0.0.1:5174
```

也可以直接打开 `index.html` 体验基础页面，但推荐使用本地服务器运行。

## 当前核心玩法

- 放置矿源、采矿机、熔炉、高温熔炉、铸造厂、组装工厂、仓库、发电机和电杆
- 拖拽物流线连接设备输入/输出端口
- 拖拽电力线连接发电机、电杆和用电设备
- 矿源拥有有限储量，当前上限为 `50000`
- 矿源拥有 3 个输出口，可并联多台采矿机
- 采矿机从矿源开采对应矿物，再输出到生产设备
- 熔炉、高温熔炉、铸造厂和组装工厂支持配方选择
- 配方选择使用分组弹窗，显示材料、产物和用途说明
- 仓库支持单资源锁定、清空后自动解锁、输入/输出口开关和出口限流
- 多槽位本地存档，可新建、覆盖、读取和删除

## 电力系统

- 发电机消耗煤矿产生电力
- 电杆可作为电力中继，并拥有 3 个输出口
- 电力线有容量限制
- 电力线负载会递归计算下游需求
- 电杆输入线负载 = 电杆自身耗电 + 下游所有输出分支耗电
- 电网监控使用动态图形仪表：
  - `< 60%`：绿色 / 稳定
  - `>= 60%`：黄色 / 高负载
  - `>= 80%`：红色 / 过载
  - `>= 100%` 或真实过载：红色 / 已超载
- 过载电力线会闪烁，高负载会在监控和警报中提示

## 物流系统

- 物流包沿连线路径移动，移动速度不随线路长度变化
- 物流监控使用 5 秒滑动平均吞吐，避免瞬时数据跳动
- 物流线健康状态：
  - 正常：蓝色
  - 高负载：黄色
  - 堵塞：红色闪烁
- 物流监控面板显示物流效率、平均吞吐、在线资源包、线路健康和最拥堵线路

## 阶段目标

游戏包含第一阶段目标链：

1. 建立铁矿采集线
2. 稳定生产铁锭
3. 接入稳定电网
4. 进入钢铁生产
5. 准备核心组件
6. 完成机械核心目标

目标完成后会奖励设备或线缆库存，引导玩家逐步扩展产线。

## 操作说明

- 点击底部设备按钮放置节点
- 拖拽节点移动位置
- 从输出口拖到输入口创建物流线
- 从黄色电力输出口拖到黄色电力输入口创建电力线
- 拖线过程中按 `Esc` 取消
- 点击已有连线选中，按 `Delete` 删除
- 点击节点后可在右侧查看状态、缓存、配方和容量
- 使用工具栏的“存档 / 读档”管理多个本地存档
